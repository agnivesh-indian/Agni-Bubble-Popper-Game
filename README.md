A simple and relaxing bubble popping game built with HTML, CSS, and JavaScript.

## How to Play

1.  **Download the files.**
2.  **Add a sound file:**
    *   Find a short, gentle "pop" sound effect.
    *   Save it as `pop.mp3` inside the `sounds/` folder.
3.  **Open `index.html`** in your web browser.
4.  Click or tap on the bubbles to pop them!
5.  Use the button in the corner to toggle the sound on or off.

## Game Features

*   **Endless Bubbles:** Bubbles continuously float up from the bottom.
*   **Satisfying Pops:** Enjoy a visual burst effect and a gentle sound with each pop.
*   **Relaxing Gameplay:** No scores, no timers, just simple popping fun.
*   **Responsive Design:** Works on both desktop and mobile devices.
*   **Sound Control:** Easily toggle the pop sound on or off.

## Optional Improvements

*   **Different Bubble Colors:** Modify the `createBubble` function in `script.js` to generate a wider variety of colors.
*   **Varying Bubble Speeds:** Adjust the `animationDuration` in `script.js` for more dynamic movement.
*   **Background Music:** Add a calming background music track to the `index.html` file with an `<audio>` tag and control it via JavaScript.
